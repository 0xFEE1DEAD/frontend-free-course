# Задание (radio). Запуск контейнера.

Что будет выведено в консоль при запуске:
```bash
podman run docker.io/hello-world
```
Выберите правильный вариант ответа:
* Ничего не будет выведено, т. к контейнер не будет запущен из-за отсутствия образа
* Будет выведено сообщение `Hi!`
* Будет выведено сообщение `Hello from Docker! ...`
* Будет выведено сообщение об ошибке `Error: unrecognized command 'podman run'`
